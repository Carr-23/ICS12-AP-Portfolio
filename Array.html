What Are Arrays?

 

Think of arrays as a school binder. In that binder you store anything you want, as long as it belongs to the same course. The papers in the binder will stay there unless you choose to move them around. Well arrays are very similar. It stores values of the same data type such as int or double. However, there are two types of arrays. Arrays and Array Lists.

Arrays: These types of arrays are used to store basic values. Such as int, String, or double. These kinds of arrays come in many shapes and sizes.

  • 1D Array: a one dimensional array allows you to store values in a linear way. This means that it will be like a list, but stretched out horizontally. These kinds of arrays are normally used when the values you do have are of the same category. Ex. Students marks in the class.
  • 2D Array: a two dimensional array allows you to store values similarly to a table. You have your columns and your rows. These types of arrays allows us to two multiple values of the same kinda that may not fall under the same catergory. Ex. One row is for students marks in the latest test and a seperate row may be for their overall average.

 

Array Lists: These types of arrays are used to store more complex values. More often than not, we use array lists to store objects from a class.
