When it comes to arrays, we typically use for loops because we are able to use the counter of the loop as a variable when dealing with arrays. However, with arraylists, we use something called a For-Each Loop

Ex.

for (ElementType tempElementName: arrayListName){

}

A for each loop repeats itself until each element in the array is called. tempElementName refers to the element in the current location in the arraylist (starts with the first element at location 0. When it repeats it then goes to the next element until all were used). The loop gets the value from arrayListName. This loop allows us to acess each element without having to actually call it by location.
