Searching

What if we don´t want to sort a list of an array, but we still want to know the locations of them? Well you´re in luck! With search algorithms we are able to search for a certain value in an array. In return we can get a boolean statement, letting us know that the value exists or an int statement, letting us know the exact location which the same value appears.

Sequential (Linear) Search: This is the most common kind of search. It checks every value in the array starting from the begining. It takes each value in the array and compares it to the array that the user is trying to find.
