Abstract Methods & Classes

Abstract classes are classes that are declared abstract. Abstract classes can only be a subclass.

An abstract class is declared without any interior code and instead ending with a semicolon.

Ex. abstract void eatFood (int food);

Any class that has abstract methods must also be declared as an abstract class.

Abstract classes are used when methods are common among multiple classes and you want to share the code among them.

The method in the subclass that will determine what happens in the abstract method must have the same name as the abstract methods
