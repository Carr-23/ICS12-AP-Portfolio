Sorting

When it comes to sorting values there are many algorithms that are capable to doing the same exact task. The difference between them is the amount of memory one may use and the efficiency. The more common algorithms are made to sort values from greatest to least or vice versa. Sorting algorithms area really useful, especially when dealinn with a large amount of values.

Insertion Sort: The way insertion sort works is that it sorts the values as it goes on. It starts of with the first 2 values and sorts it. Then then first 3 so on so forth. It continues this process once it reaches the last value and finishes sorting it.

CockTail Shaker Sort: Cocktail shaker sort works opposite to insertion sort. Instead of having the array increase it decreases. Instead of sorting Everything as it goes, cocktail only sorts 2 values at a time. Once it finishes checking every pair of values and it reaches the end, the array size will decrease and then it will go backwards, moving the smallest values down. Essentially the sorting algorithm will go up and down the array as it gets smaller. This motion mimics that of a cocktail shaker.
